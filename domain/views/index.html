Welcome to the API domain!

This site serves to provide API endpoints to the "askme clone" frontend project. It allows the webpage to authenticate users by logging them in or create them if they don't have an account. Users that are reguistered can be asked questions anonymously by other users.

This site also provides access to the database resources, though some of them are restricted to authenticated or even authorised users.

If they wish to reply, their response with the question will appear on their profile. Only question that have been answered can appear on the profile.
